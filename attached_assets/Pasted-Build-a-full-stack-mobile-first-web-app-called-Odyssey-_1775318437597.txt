Build a full-stack mobile-first web app called Odyssey.

Odyssey is an AI-powered behavioral learning platform where parents and teachers create structured learning experiences (“Adventures”) for children, supported by a coin-based reward system.

The system MUST be personalized using a Learner Profile, and all AI features must adapt based on that profile.

🧠 CORE SYSTEM LOOP

Child completes step → earns coins → wallet updates → reward progress updates → reward unlocks → reward redeemed → behavior reinforced

This loop must work end-to-end with real data persistence.

⚙️ TECH STACK

Frontend:

React (Vite)
Tailwind CSS
Zustand (state)

Backend:

Supabase (preferred) OR Node.js + Express

AI:

OpenAI API (GPT-4o or latest)
